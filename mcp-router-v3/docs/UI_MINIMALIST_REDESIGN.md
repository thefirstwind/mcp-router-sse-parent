# MCP Router v3 UI 简约未来风格重设计

## 📋 设计理念

参考 MCP Inspector 的简约未来风格，采用极简设计原则，去掉所有花哨的装饰效果，专注于**清晰、专业、现代**的视觉呈现。

## 🎨 设计原则

### 1. 极简主义
- **去除装饰**：去掉渐变背景、花哨动画、表情符号
- **简洁配色**：中性色调为主，少量强调色
- **清晰层次**：通过边框、间距、字体大小区分层次

### 2. 专业代码编辑器风格
- **GitHub 风格**：参考 GitHub 的简洁设计
- **VS Code 风格**：参考代码编辑器的专业感
- **MCP Inspector 风格**：参考官方工具的简约设计

### 3. 未来感
- **扁平化设计**：去除立体效果
- **微妙交互**：轻微的悬停效果
- **现代字体**：系统字体栈，清晰易读

## 🎯 核心改进

### 1. 颜色系统（重新设计）

#### 主色调
- **背景**：白色 (#ffffff) + 浅灰 (#f8f9fa)
- **文字**：深灰 (#24292e) + 中灰 (#586069) + 浅灰 (#959da5)
- **强调色**：蓝色 (#0366d6) - 仅用于重要元素
- **边框**：浅灰 (#e1e4e8)

#### 功能色
- **成功**：绿色 (#28a745)
- **错误**：红色 (#d73a49)
- **警告**：橙色 (#ffa500)
- **信息**：蓝色 (#0366d6)

### 2. 布局优化

#### 头部
- **简洁设计**：白色背景 + 底部边框
- **固定定位**：sticky 头部，滚动时保持可见
- **按钮样式**：简洁的边框按钮

#### 卡片
- **扁平设计**：白色背景 + 细边框
- **无阴影**：去除阴影，使用边框
- **圆角**：6px 小圆角

#### 表格
- **清晰边框**：细边框分隔
- **简洁表头**：浅灰背景
- **行悬停**：轻微背景色变化

### 3. 交互优化

#### 标签页
- **简洁设计**：底部边框指示器
- **无背景渐变**：纯色背景
- **激活状态**：蓝色下划线

#### 按钮
- **边框样式**：细边框 + 背景色
- **悬停效果**：轻微背景色变化
- **无动画**：去除所有动画效果

#### 徽章
- **小尺寸**：紧凑设计
- **纯色背景**：无渐变
- **清晰文字**：白色文字

### 4. 代码块优化

#### JSON 展示
- **浅色背景**：GitHub 风格的代码块
- **边框设计**：细边框
- **语法高亮**：GitHub 风格的配色

#### 代码字体
- **等宽字体**：SFMono-Regular, Consolas
- **合适字号**：12px
- **清晰行高**：1.6

### 5. 去除的元素

- ❌ 渐变背景
- ❌ 花哨动画
- ❌ 表情符号
- ❌ 阴影效果
- ❌ 立体感
- ❌ 装饰性元素
